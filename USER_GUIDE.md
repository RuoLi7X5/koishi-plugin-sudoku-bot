## 基础功能

### 🎮 游戏流程

**开始游戏**
```
> 数独开始
```
- 机器人生成数独盘面图片
- 图片四周有留白，底部显示难度
- 共8题，每题限时30秒（可配置）
- 直接输入1-9抢答

**答题规则**
- ✅ 答对：获得基础分10分 + 连击加分
- ❌ 答错：扣除5分
- ⏰ 超时：无人答对时公布答案

**提前结束** ⭐ NEW
```
> 数独结束
```
- 任何人都可以使用（无需管理员权限）
- 显示已完成题目数（例如："已完成 3/8 题"）
- 进行完整结算（排行榜、积分、成就、答案图片）

**游戏结束**
- 显示本局排行榜
- 显示完整答案图片（含难度标注）
- 评选MVP（👑标识）
- 自动检测成就解锁

## 难度系统

### ⭐ 7档难度控制

数独机器人提供 **7档精细难度控制**，混合使用两个高质量生成器（sudoku-gen 和 @forfuns/sudoku）。

#### 难度档位

| 档位 | 难度 | 来源 | 给定数 | 适用人群 | 生成速度 |
|------|------|------|--------|----------|----------|
| 1 | 简单 | sudoku-gen | 36-39 | 新手入门 | ⚡⚡⚡ 极快 |
| 2 | 较易 | sudoku-gen | 30-34 | 休闲娱乐 | ⚡⚡⚡ 极快 |
| 3 | 中等 | @forfuns | 35-40 | 日常挑战 | ⚡⚡ 较快 |
| 4 | 中等+ | sudoku-gen | 26-30 | 进阶练习 | ⚡⚡⚡ 极快 |
| 5 | 困难 | @forfuns | 30-35 | 高手对决 | ⚡⚡ 较快 |
| 6 | 困难+ | sudoku-gen | 22-26 | 专家级别 | ⚡⚡⚡ 极快 |
| 7 | 极难 | @forfuns | 23-27 | 极限挑战 | ⚡ 较慢(2-6秒) |

**默认难度**：档位 2（较易）⭐

---

### 🎮 使用方式

#### 方式1：设置全局难度

使用 `难度` 命令设置默认难度，之后所有游戏都使用这个难度：

```
> 难度 5
已设置难度为：困难（级别 5）

> 数独开始
[使用难度5生成题目]

> 数独开始
[仍然使用难度5]
```

**特点**：
- ✅ 一次设置，持久有效
- ✅ 显示难度来源信息
- ✅ 适合固定难度的长期游戏

---

#### 方式2：临时指定难度

在开始游戏时直接指定难度，**仅本局有效**：

```
> 难度 5
已设置难度为：困难（级别 5）

> 数独开始 3
[临时使用难度3，全局设置不变]

> 数独开始
[回到难度5]
```

**特点**：
- ✅ 灵活调整单局难度
- ✅ 不影响全局设置
- ✅ 适合临时测试或挑战

---

#### 方式3：使用默认难度

首次使用或未设置时，使用默认难度2：

```
> 数独开始
[使用默认难度2（较易）]
```

---

### 🎯 优先级规则

```
临时参数 > 全局设置 > 默认值(2)
```

**示例理解**：

| 全局设置 | 开始命令 | 实际难度 | 说明 |
|----------|----------|----------|------|
| 未设置 | `数独开始` | 2 | 使用默认值 |
| 难度 5 | `数独开始` | 5 | 使用全局设置 |
| 难度 5 | `数独开始 3` | 3 | 临时参数优先 |
| 难度 5 | `数独开始` | 5 | 恢复全局设置 |

### ⚠️ 注意事项

1. **极难难度（档位7）**
   - 生成较慢（2-6秒）
   - 建议提前告知玩家等待
   - 题目质量最高

2. **游戏中无法修改**
   - 必须先结束当前游戏
   - 再用 `难度` 命令设置

3. **难度范围限制**
   - 必须在 1-7 之间
   - 超出范围会提示错误

## 配置选项

```yaml
# 命令配置
commandStart: "数独开始"
commandStop: "数独结束"
commandDifficulty: "难度"  # ⭐ NEW
commandScore: "积分"
commandRank: "数独排行"
commandProgress: "游戏进度"
commandExchange: "兑换"

# 游戏配置
timeout: 30                       # 每题超时（秒）
rounds: 8                         # 每轮题目数量
baseScore: 10                     # 答对基础分
penalty: 5                        # 答错扣分
streakBonus: 1                    # 连击额外加分

# 其他配置
difficulty: 2                     # 默认难度（1-7）⭐ NEW
titleDuration: 7                  # 荣誉头衔有效期（天）
```

**难度说明（1-7档）**：
- 1: 简单 (sudoku-gen, 36-39个数)
- 2: 较易 (sudoku-gen, 30-34个数) ⭐ 默认
- 3: 中等 (@forfuns, 35-40个数)
- 4: 中等+ (sudoku-gen, 26-30个数)
- 5: 困难 (@forfuns, 30-35个数)
- 6: 困难+ (sudoku-gen, 22-26个数)
- 7: 极难 (@forfuns, 23-27个数)

### 🎮 完整游戏流程（含难度设置和荣誉头衔播报）

```
玩家A: 难度 5
```

或者直接在开始时指定：
```
玩家A: 数独开始 3
机器人: [临时使用难度3，不改变全局设置]
```

```
玩家A: 数独开始

```

```
玩家B: 3
```

```
机器人: emmm...这题有这么难吗？正确答案是 4
```
