# koishi-plugin-sudoku-bot

[![npm](https://img.shields.io/npm/v/koishi-plugin-sudoku-bot?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-sudoku-bot)

在群聊中发起数独抢答游戏，统计玩家答题数据，支持积分、成就、头衔等完整养成系统。

---

## 功能特性

- **数独抢答**：自动生成数独题目并渲染为图片，玩家抢答指定格子的数字
- **积分系统**：答对得分，连续答对有连击加成，答错扣分
- **成就系统**：完成各类里程碑自动解锁成就并奖励积分
- **头衔系统**：荣誉头衔（周期性颁发）+ 积分兑换头衔
- **排行榜**：支持积分、答对数、参与数、正确率多维度排名
- **MVP 机制**：每局结束后评选本局得分最高玩家
- **嘲讽语库**：超时未答或答错时随机触发趣味提示语

---

## 安装

```bash
npm install koishi-plugin-sudoku-bot
# 或
yarn add koishi-plugin-sudoku-bot
```

在 `koishi.yml` 中启用插件：

```yaml
plugins:
  sudoku-bot:
    difficulty: medium
    rounds: 8
    timeout: 30
```

## 指令说明

| 指令（默认） | 说明 | 权限 |
|---|---|---|
| `数独开始` | 在当前群聊发起一局数独游戏 | 所有人 |
| `数独结束` | 强制终止当前游戏 | 管理员（authority ≥ 4） |
| `积分` | 查看自己的数独档案与统计数据 | 所有人 |
| `兑换 <头衔名>` | 花费积分兑换指定头衔 | 所有人 |
| `数独排行 [类型]` | 查看排行榜 | 所有人 |
| `游戏进度` | 查看当前游戏的实时进度 | 所有人 |

### 排行榜类型

| 参数 | 榜单 | 备注 |
|---|---|---|
| `score`（默认） | 积分榜 | 按总积分排名 |
| `correct` | 答对榜 | 按历史答对题数排名 |
| `rounds` | 参与榜 | 按参与总局数排名 |
| `rate` | 正确率榜 | 需答题数 ≥ 5 方可上榜 |

### 答题方式

游戏进行时，发送 `1`~`9` 的单个数字即可参与抢答，无需额外指令前缀。

## 游戏玩法

1. 发送「数独开始」，Bot 发送当前数独棋盘图片
2. Bot 依次提问，例如：`第1题：C3格应该填什么？`
3. 玩家在超时前发送 `1`~`9` 抢答
4. 答对得分并进入下一题，答错扣分且继续等待其他人作答
5. 超时无人答对时，Bot 公布答案并进入下一题
6. 所有题目结束后，展示本局排行榜并评选 MVP

## 积分规则

| 事件 | 积分变化 |
|---|---|
| 答对 | `基础分 + (连击数 - 1) × 连击加成` |
| 答错 | `-扣分值` |
| 解锁成就 | 按成就奖励加分 |
| 兑换头衔 | 按头衔价格扣分 |

## 头衔系统

### 荣誉头衔（自动颁发）

每局游戏结束后，Bot 自动计算各维度 TOP 1 并颁发限时荣誉头衔（默认有效期 7 天，可配置）：

| 头衔 | 评选标准 |
|---|---|
| 积分王者 | 总积分最高 |
| 答题达人 | 答对题数最多 |
| 参与之星 | 参与局数最多 |
| 活跃先锋 | 发起游戏次数最多 |
| 准确之神 | 正确率最高（需参与 ≥ 10 局） |

### 积分兑换头衔

| 头衔 | 价格 | 有效期 |
|---|---|---|
| 数独学徒 | 100 积分 | 7 天 |
| 解题高手 | 500 积分 | 30 天 |
| 终局大师 | 2000 积分 | 365 天 |

## 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `commandStart` | string | `数独开始` | 开始游戏的指令 |
| `commandStop` | string | `数独结束` | 强制结束游戏的指令（管理员） |
| `commandScore` | string | `积分` | 查看积分档案的指令 |
| `commandExchange` | string | `兑换` | 兑换头衔的指令 |
| `commandRank` | string | `数独排行` | 查看排行榜的指令 |
| `commandProgress` | string | `游戏进度` | 查看游戏进度的指令 |
| `timeout` | number | `30` | 每题超时时间（秒） |
| `rounds` | number | `8` | 每局题目数量 |
| `baseScore` | number | `10` | 答对基础分 |
| `penalty` | number | `5` | 答错扣分 |
| `streakBonus` | number | `1` | 连续答对每题额外加分 |
| `difficulty` | string | `medium` | 难度：`easy` / `medium` / `hard` |
| `titleDuration` | number | `7` | 荣誉头衔有效期（天） |

---

## 注意事项

- 游戏只能在**群聊**中进行，私聊无效
- 每个群同一时间只能有一局游戏进行
- 强制结束命令需要用户 authority ≥ 4
- 正确率榜要求累计答题数 ≥ 5 才会显示
